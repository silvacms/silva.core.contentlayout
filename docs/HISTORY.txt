Changelog
=========

1.0.6 (2011-10-20)
------------------

- add styles for zeam collection widget

1.0.5 (2011-09-28)
------------------

- really release, correcting for accidental release of ncolumn-changes
  branch (no real changes in this release)

1.0.4 (2011-09-19)
------------------

- adjusted the default tinymce configuration

- fix bug in TinyMCERichText adapter's to_public, to allow empty strings
  to be rendered
  
- add support for third-part validators - to validate invariants of IParts.
  (invariants are hard to implement for external sources with formulator forms,
  using IPartInvariantValidator adapters are a way to mediate this)

- adjust the width of the editor/add panels

1.0.3 (2011-09-09)
------------------

- add missing custom rich text edit widget, work around import bug

- fix preview button, so it points to the correct ++preview++ path 
  (++preview++ is not on IFolder, just IPublication, so add after the
  first path component -- assumes the first is a publication)

- make cursor a pointer when hovering over ES options on the 'add to page'
  interface

1.0.2 (2011-08-17)
------------------

- enable sticky service tab button for all containers (from just publication)

1.0.1 (2011-08-10)
------------------

- fixed bug when acquiring the nearest IStickyContentService local utility

- created a default content template for Ghosts, so that their rendering
  is isolated from the rest.

1.0 (2011-07-07)

- Initial release
